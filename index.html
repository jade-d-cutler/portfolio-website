My Projects

Password Manager - This Password Manager, built in Python, offers a simple and secure way to manage your credentials. It features fields to enter your website, username/email, and password. There is a button that will generate a randomized password created by the program. There is a search button that functions by you entering a website, clicking the button, and it will display your credentials for that site. Finally, the "Add" button allows you to securely store your credentials for a site in a JSON file on your computer.

Tasty Table Recipe Website - Tasty Table is a user-friendly food recipe website designed to help people discover delicious meals they can cook at home. Featuring 20 carefully curated recipes, the main page presents each recipe as a card for easy browsing. Clicking on a recipe card takes users to a dedicated page with detailed instructions and ingredients. The website also includes an "About" page that shares the mission behind Tasty Table and a "Contact" page for users to get in touch with the team.

Turtle Crossing Game - This Python Turtle-based game is simple: guide your character across the road, avoiding cars that move at increasing speeds with each level. As the difficulty ramps up, players must time their moves carefully to avoid getting hit.

Flashcards - This flashcard app helps users learn French vocabulary by displaying a French word on the front of each card, which flips after a three-second pause to reveal the English translation. Below the card, two buttons shaped like an X and a green checkmark allow users to track their progress. Clicking the X shows the next card, while clicking the checkmark removes that word from the study loop, ensuring only unfamiliar words are repeated.

Pomodor Tomato Timer - This Pomodoro Timer is designed to enhance productivity using the popular Pomodoro technique. The timer starts with a 25-minute work session, followed by a 5-minute break. This cycle repeats five times, with a longer 10-minute break at the end of the fifth cycle. A restart button allows users to reset the timer and start the process over.
